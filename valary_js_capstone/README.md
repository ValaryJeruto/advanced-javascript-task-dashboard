# Advanced JavaScript Task Dashboard

Prepared by: Valary Jeruto

## Overview
This project is a browser-based JavaScript application created for a capstone toolkit. It demonstrates more advanced beginner-friendly JavaScript concepts, including:

- DOM manipulation
- form validation
- event handling
- array methods
- search, filter, and sort features
- localStorage persistence
- JSON export
- edit, delete, and complete actions

## Files
- `index.html` — app layout
- `styles.css` — visual styling
- `app.js` — JavaScript logic
- `javascript_capstone_advanced.Rmd` — capstone documentation in R Markdown format

## How to Run
1. Download or unzip the project folder.
2. Open the folder in VS Code.
3. Double-click `index.html` or open it in your browser.
4. Add tasks, filter them, edit them, and export them as JSON.

## What Makes This Project Advanced
Compared with a basic "Hello World" example, this app uses:

- a JavaScript class to organize logic
- localStorage to save data between sessions
- reusable render functions
- multiple UI controls working together
- user feedback messages
- dynamic task cards generated from a template

## Suggested Demo Flow
1. Create a few tasks
2. Mark one as done
3. Search by keyword
4. Filter overdue tasks
5. Sort by priority
6. Export the task list as JSON

## Notes
The `.Rmd` file is included as the capstone toolkit source. It is ready to open in RStudio or Posit if you want to render it later.
